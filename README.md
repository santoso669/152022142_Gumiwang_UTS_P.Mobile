Aplikasi UTS Pemrograman Mobile – Flutter Repository: 152022142_Gumiwang_UTS_P.
=
Mobile Dikembangkan untuk memenuhi tugas Ujian Tengah Semester (UTS) Mata Kuliah Pemrograman Mobile, Semester 7.

Aplikasi ini terdiri dari beberapa halaman utama yang dihubungkan melalui Bottom Navigation Bar, berisi fitur–fitur dasar pemrograman Flutter seperti widget, navigasi, layouting, media, hingga penggunaan package.

Deskripsi Aplikasi yang Dikembangkan
=
Aplikasi UTS ini dibuat menggunakan framework Flutter, dengan tujuan untuk mempraktikkan pemahaman mengenai widget, layouting, navigasi, serta penggunaan package eksternal.

Fitur–fitur yang dikembangkan meliputi:

• Splash Screen 5 detik, yang berisikan (judul aplikasi, 
foto anda, NIM, nama) 

• Navigasi menggunakan BottomNavigationBar yang sudah terdiri dari Biodata, Kontak, Kalkulator, Cuaca, dan Berita. Semua menu-menu tersebut sudah dapat dipilih, dan menggunakan konsep Fragment. 

• Halaman Biodata terdiri dari informasi teks dan image foto profile, ada juga isian informasi tertentu seperti 
dropdown, radio button, teks, dan calendar semuanya sudah bisa melakukan input data. 

• Halaman Kontak sudah terdiri dari daftar telepon yang terdaftar secara statis, layout user sudah terdiri dari teks dan circle image, sudah ada 15 kontak telepon. dan Data bersifat statis yang berasal dari variable. 

• Kalkulator dengan desain menyerupai kalkulator smartphone

• Halaman Cuaca, data sudah statis dan sudah ada informasi suhu dan kelembapan serta gambar animasi kondisi cuaca dengan video .mp4 dari asset menggunakan package video_player, 

• Halaman Berita yang berisi 3 berita asli lengkap dengan gambar, judul, dan isi ringkas

Aplikasi ini dibangun untuk melatih kemampuan dalam merancang antarmuka, mengelola state sederhana, menampilkan media, dan menyusun struktur proyek Flutter secara benar.
